# Full i18n Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Localize Flashot across English, Simplified Chinese, and Traditional Chinese with Taiwan wording, removing the system language option and extending localization to Rust-native visible UI.

**Architecture:** Keep the lightweight dictionary architecture, make English the typed source of truth, add `zh-TW`, and wire current language through visible frontend surfaces. Mirror the same language enum in Rust and add a small native i18n module for tray labels and utility window titles.

**Tech Stack:** React, TypeScript, Vitest, React Testing Library, Tauri 2, Rust serde tests.
