# Settings

Flashot's settings window lets you customize every aspect of the app. Access it from the tray icon menu.

The settings are organized into three tabs: **General**, **Appearance**, and **Shortcuts**.

## General

### Default Save Directory

Sets the folder where screenshots are saved when you use **Save**. Click the folder icon to browse and choose a different location. The default is `~/Pictures/Flashot`.

### Launch at Login

Enable this to start Flashot automatically when you log into your computer. Flashot runs quietly in the background until you trigger a capture.

### Automatic Updates

Controls how Flashot checks for and installs updates:

- **Auto-check for updates** — When enabled, Flashot periodically checks for new versions at the configured interval (default: every 24 hours, configurable from 1–168 hours).
- **Allow beta updates** — When enabled, Flashot checks for beta/prerelease versions first, then falls back to stable releases. Useful for testing upcoming features.

## Appearance

### Language

Flashot supports three languages:

| Language | Code |
|----------|------|
| English | `en` |
| Simplified Chinese | `zh-CN` |
| Traditional Chinese | `zh-TW` |

The language setting affects all UI text including settings, annotation tooltips, and the about window.

### Theme

Three modes available:

- **System** — Follows your OS-level appearance setting (light / dark)
- **Light** — Always light
- **Dark** — Always dark

### Accent Color

Choose from five accent colors that tint selection boxes, toolbar highlights, and pin glows:

| Color | Preview |
|-------|---------|
| Cyan | `#22d3ee` |
| Rose | `#f43f5e` |
| Amber | `#f59e0b` |
| Emerald | `#10b981` |
| Violet | `#8b5cf6` |

## Shortcuts

Three global hotkeys can be customized:

| Function | Default (macOS) | Default (Windows/Linux) | Description |
|----------|-----------------|-------------------------|-------------|
| Region capture | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | Opens the overlay for selection |
| Full screen | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> | Quick shot — entire screen |
| Active window | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> | Quick shot — frontmost window |

To change a shortcut:

1. Click the **shortcut field** next to the action.
2. Press the desired key combination on your keyboard.
3. The field updates to show your new shortcut.

Flashot will warn you if two shortcuts conflict. Shortcuts update live — no restart required.

### Reset to Defaults

Click the **Reset** button at the bottom of the settings window to restore all default values.
