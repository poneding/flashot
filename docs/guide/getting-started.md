# Getting Started

Once Flashot is installed and running (look for the lightning bolt icon in your system tray / menu bar), you're ready to capture.

## Default Shortcuts

| Action | macOS | Windows / Linux |
|--------|-------|-----------------|
| Region capture | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> |
| Full screen quick shot | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> |
| Active window quick shot | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> |

## Your First Capture

1. Press **<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>** (macOS) or **<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>** (Windows/Linux).
2. The screen freezes and the overlay appears.
3. **Click and drag** to select a region, or **click once** on a window to auto-select its bounds.
4. Release the mouse to confirm the selection.

![Flashot capture demo](/images/demo-01.png)

### After Selection

Once you've committed a selection, a toolbar appears with options:

- **Copy** — Copy the screenshot to your clipboard (<kbd>Cmd</kbd> + <kbd>C</kbd>)
- **Save** — Save as a PNG file (<kbd>Cmd</kbd> + <kbd>S</kbd>)
- **Pin** — Pin as a floating window
- **Scroll** — Start a scrolling capture session
- **Close** — Cancel (<kbd>Esc</kbd>)

You can also **resize** the selection by dragging its handles, or **move** it by dragging from inside the selected area.

## Quick Shots

Quick shots bypass the overlay and copy the result immediately:

- **Full screen** — Captures your entire primary display
- **Active window** — Captures the frontmost window

## Annotating

When a selection is committed, an annotation toolbar appears above the overlay. Use it to draw arrows, rectangles, text, blur sensitive information, and more. Switch between tools freely — each tool remembers its own style.

## Cancelling

Press <kbd>Esc</kbd> at any time during a capture to cancel and return to the desktop.
