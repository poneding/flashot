# 圖釘

**圖釘**功能將截圖轉為始終置頂的浮動視窗。適用於：

- 在其他應用中工作時參考截圖
- 並排比較多張截圖
- 在簡報中保持標註後的截圖可見

## 釘截圖

1. 擷取一個區域。
2. 點選操作工具列中的**圖釘**按鈕。
3. 截圖在新浮動視窗中打開。

## 控制項

懸停在釘住的截圖上（或取得焦點）時顯示控制項。控制項位於釘住視窗的右側：

| 控制項 | 操作 |
|---------|--------|
| ✏️ **編輯** | 進入標註模式，加入或修改標註 |
| 🖼️ **調整** | 打開影像調整面板 |
| 🔍 **縮放** | 打開縮放比例選擇器（50%–300%） |
| ✕ **關閉** | 關閉釘住視窗 |
| 💾 **儲存** | 將釘住的截圖儲存為 PNG |
| 📋 **複製** | 複製到剪貼簿（帶勾選確認動畫） |

### 縮放

- **滑鼠滾輪** — 向上/向下滾動以 5% 為步長放大/縮小
- <kbd>Cmd</kbd> + <kbd>+</kbd> — 放大
- <kbd>Cmd</kbd> + <kbd>-</kbd> — 縮小
- <kbd>Cmd</kbd> + <kbd>0</kbd> — 重設為 100%

目前縮放百分比短暫顯示在左上角。

### 移動

在釘住的影像上點選並拖拽以重新定位視窗。

### 關閉

- 點選**關閉**按鈕
- 按 <kbd>Esc</kbd>
- **雙擊**釘住影像上的任意位置

## 編輯釘住的截圖

點選**編輯**按鈕進入釘住截圖的標註模式：

1. 完整的標註工具列可用。
2. 在釘住的影像上繪製標註。
3. 按 <kbd>Esc</kbd> 或再次點選**編輯**儲存並退出。

儲存或複製時，標註會併入輸出。標註也作為獨立圖層持久儲存，以便後續重新編輯。

## 圖釘上的影像調整

調整面板也適用於釘住的截圖。調整亮度、對比度、飽和度或啟用灰階 —— 效果在儲存/複製時套用。

## 鍵盤快速鍵

| 按鍵 | 操作 |
|-----|--------|
| <kbd>Esc</kbd> | 退出編輯模式或關閉圖釘 |
| <kbd>E</kbd> | 進入標註編輯模式 |
| <kbd>Cmd</kbd> + <kbd>C</kbd> | 複製圖釘到剪貼簿 |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | 將圖釘儲存為 PNG |
| <kbd>Cmd</kbd> + 滾輪 | 放大/縮小 |
