# 螢幕擷取

Flashot 提供三種擷取模式，每種透過全域快速鍵觸發。

## 區域擷取

主要的擷取模式。按下區域擷取快速鍵，畫面凍結並進入覆蓋層介面。

1. 螢幕變暗，所有顯示器顯示凍結的幀畫面。
2. **點選並拖動**繪製選取矩形。
3. **放開**滑鼠確認選取。
4. 標註工具列和操作工具列出現。

### 智慧視窗偵測

將滑鼠懸停在任意可見視窗上 —— Flashot 會自動高亮其邊界。**點選**高亮的視窗即可立即選中，無需手動拖拽。

該功能支援跨顯示器工作，使用 Z 軸點選測試識別游標下的最頂層視窗。

### 選取操作

選定區域後：

- **調整大小** — 拖拽 8 個手柄（角 + 邊）調整選取區域。
- **移動** — 在選取區域內點選並拖拽以重新定位。
- **圓角** — 透過工具列滑動條調整截圖邊角圓度。

## 全螢幕快捷擷取

按下全螢幕快速鍵，立即擷取**主顯示器**的畫面。截圖自動複製到剪貼簿 —— 無需覆蓋層，無需互動。

在多顯示器環境中，預設快速鍵只擷取主顯示器。如需擷取其他顯示器，請使用區域擷取。

## 視窗快捷擷取

按下視窗快速鍵，立即擷取主顯示器上**最前面的視窗**。與全螢幕快捷擷取一樣，結果自動複製到剪貼簿。

適用於擷取對話框、獨立應用視窗或任何需要手動選擇的介面元素。

## 輸出

### 複製到剪貼簿

區域擷取後，點選**複製**（<kbd>Cmd</kbd> + <kbd>C</kbd>）將截圖傳送到剪貼簿。您可以直接貼到任何應用中（文件、聊天、圖片編輯器）。

快捷擷取會自動複製 —— 無需額外操作。

### 儲存為 PNG

點選**儲存**（<kbd>Cmd</kbd> + <kbd>S</kbd>）將截圖儲存到設定的預設目錄。首次儲存會彈出檔案對話框；後續儲存使用上次使用的目錄。

預設儲存位置為 `~/Pictures/Flashot/`。可在[設定](/zh-TW/guide/settings)中修改。

### 釘為浮動視窗

點選**圖釘**將截圖轉為始終置頂的浮動視窗。釘住的截圖可標註、調整、縮放和調整大小 —— 詳見[圖釘](/zh-TW/features/pin)功能頁面。

## 多顯示器支援

觸發區域擷取時，Flashot **同時擷取所有連接的顯示器**。每台顯示器獲得獨立的覆蓋層視窗，包含正確的幀畫面和縮放因子。您可以在任何顯示器上無縫選擇 —— 游標位置決定哪個覆蓋層處於互動狀態。

擷取系統處理：

- 不同解析度和縮放因子的顯示器
- 非規則的顯示器排列（並排、上下堆疊、混合）
- 完整的顯示器資訊（位置、尺寸、縮放因子）
