# 影像調整

在儲存或複製截圖之前，您可以即時套用影像調整。這些調整在擷取時即時預覽，並會併入輸出檔案。

## 調整控制

從截圖工具列中打開調整面板（尋找影像圖示）。提供四個控制項：

### 亮度

調整影像的整體明暗程度。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 明顯變亮 |
| `+1` 到 `+49` | 略微變亮 |
| `0` | 無變化 |
| `-1` 到 `-49` | 略微變暗 |
| `-50` 到 `-100` | 明顯變暗 |

### 對比度

控制亮部和暗部之間的差異。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 高對比度 —— 鮮明、清晰 |
| `+1` 到 `+49` | 略微提高對比度 |
| `0` | 無變化 |
| `-1` 到 `-49` | 略微降低對比度 |
| `-50` 到 `-100` | 低對比度 —— 平淡、柔和 |

### 飽和度

控制色彩強度。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 鮮豔、過飽和的色彩 |
| `+1` 到 `+49` | 略微更生動 |
| `0` | 無變化 |
| `-1` 到 `-49` | 略微去飽和 |
| `-50` 到 `-100` | 高度去飽和 |
| **灰階開啟** | 完全去飽和（覆蓋滑動條） |

### 灰階

一個開關，移除影像中的所有顏色。啟用時覆蓋飽和度滑動條。

## 使用說明

- 調整在凍結層上**即時預覽**。
- 調整同時影響**剪貼簿複製**和**儲存的檔案**。
- 調整是**非破壞性的** —— 原始幀保留在記憶體中。
- 釘住的截圖視窗也支援調整，以便在匯出前精細調整。

## 鍵盤快速鍵

調整功能沒有直接的鍵盤快速鍵。請使用面板上的控制項或單獨重設每個滑動條。
