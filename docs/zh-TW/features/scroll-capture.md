# 滾動截圖

滾動截圖讓您可以捕捉超出可視區域的內容 —— 如長網頁、文件或聊天記錄 —— 透過自動拼接多張畫面實現。

## 工作原理

1. 在螢幕上**選擇一個區域**。
2. 點選操作工具列中的**滾動**按鈕（注意：所選區域高度必須至少 100px）。
3. 出現半透明覆蓋層，提示您**滾動內容**。
4. 開始滾動目標視窗 —— Flashot 會隨著滾動自動捕捉幀。
5. 狀態藥丸顯示目前的**幀數**和**總高度**。
6. 點選右下角的**綠色勾選按鈕**（✓）完成擷取。

捕捉到的幀會被拼接成一幅完整的長圖，然後儲存或複製。

## 滾動 Chrome 介面

滾動截圖過程中，會出現一個「滾動 Chrome」視窗，包含：

- **即時預覽** — 捕捉到的幀垂直堆疊，帶平滑過渡動畫
- **狀態藥丸** — 底部中央顯示 `X 幀 · Y px`
- **完成按鈕** — 綠色勾選按鈕結束擷取
- **自動停止** — 達到最大高度時自動結束工作階段

## 技術細節

- 隨著滾動，每隔幾百毫秒捕捉一幀。
- 每幀與前一幀使用評分演算法比較，偵測新內容。
- 僅保留包含足夠新內容的幀（去重）。
- 使用基於 NCC（歸一化互相關）評分的接縫檢測演算法，在重疊內容中尋找最佳拼接位置。

## 提示

- **平穩滾動** — 穩定、中速的滾動產生最佳的拼接效果。
- **選擇合適的區域** — 擷取區域應覆蓋內容區域，但如果可能，排除固定的頁首/頁尾。
- 最小選擇高度為 100px。
- 最大拼接高度取決於可用記憶體和內容。
