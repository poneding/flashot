# 標註工具

確認選區後，Flashot 進入標註模式。標註工具列出現在選定區域上方，提供 **13 種工具**用於標記截圖。

## 工具一覽

| 工具 | 圖示 | 用途 |
|------|------|---------|
| 矩形 | `□` | 繪製描邊或填滿矩形 |
| 橢圓 | `○` | 繪製描邊或填滿橢圓 |
| 箭頭 | `→` | 附可調箭頭大小的方向指示 |
| 直線 | `—` | 可自訂描邊的直線 |
| 畫筆 | `✏️` | 可變粗細的自由手繪 |
| 文字 | `T` | 加入文字標籤，支援字型、大小和顏色選擇 |
| 序號標記 | `#` | 帶編號的標註，適合依序標記 |
| 螢光筆 | `🖍️` | 半透明高亮 —— 兩種模式：自由手繪傾斜或直線 |
| 馬賽克 | `●` | 對敏感資訊（文字、人臉、號碼）進行像素化模糊處理 |
| 放大鏡 | `🔍` | 放大截圖的某個區域以查看細節 |
| 聚光燈 | `🔦` | 除焦點區域（圓形或矩形）外全部變暗 |
| 測量 | `📏` | 測量像素距離並顯示尺寸 |
| 橡皮擦 | `🧹` | 點選即可刪除單個標註物件 |

每個工具獨立記憶自己的樣式設定（描邊顏色、填滿顏色、不透明度、粗細、字型）—— 每個工具最後使用的樣式會被記住。

## 繪製流程

1. 點選標註工具列中的圖示**選擇工具**。
2. 在截圖上**點選並拖動**建立標註物件。
3. **放開**將其放置在影像上。
4. **點選已有物件**選中它，在工具列下方出現的屬性面板中調整屬性。

標註工具列是**可拖動的** —— 抓住左側的拖拽手柄即可將其移動到螢幕上任一位置。

## 屬性面板

選中標註物件後，屬性面板顯示其可調屬性：

| 屬性 | 適用工具 |
|----------|------------|
| 描邊顏色 | 除橡皮擦外的所有工具 |
| 描邊寬度 | 矩形、橢圓、箭頭、直線、畫筆 |
| 填滿顏色 | 矩形、橢圓 |
| 填滿不透明度 | 矩形、橢圓 |
| 字型 | 文字 |
| 字型大小 | 文字 |
| 箭頭大小 | 箭頭 |
| 模糊半徑 | 馬賽克（匯出時套用高斯模糊） |
| 標記編號 | 序號標記 |

## 復原 / 重做

- **復原** — <kbd>Cmd</kbd> + <kbd>Z</kbd> / <kbd>Ctrl</kbd> + <kbd>Z</kbd>
- **重做** — <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> / <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd>

## 刪除

選中物件後按 **<kbd>Delete</kbd>** 或 **<kbd>Backspace</kbd>**，或使用橡皮擦工具點選物件將其移除。

## 工具提示

每個工具按鈕懸停時顯示描述性提示。工具切換沒有鍵盤快速鍵 —— 透過點選工具列選擇工具。

## 標註持久化

**複製**或**儲存**時，標註會被柵格化並合併到輸出影像中。對於釘住的截圖，標註會作為獨立圖層儲存，可在圖釘編輯模式中重新編輯。
