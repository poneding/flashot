# 取色器

Flashot 內建取色器，讓您直接從截圖中取樣顏色。

## 使用方法

1. 擷取區域並確認選取。
2. 將游標懸停在截圖區域上。
3. 取色器面板在覆蓋層處於**懸停**或**已確認**模式時自動顯示。
4. 移動游標查看即時顏色值更新。

## 顏色格式

取色器支援三種格式，按 <kbd>X</kbd> 鍵切換：

| 格式 | 範例 | 說明 |
|--------|---------|-------------|
| **HEX** | `#22d3ee` | 標準十六進位表示 |
| **RGB** | `rgb(34, 211, 238)` | 紅、綠、藍（0–255） |
| **HSL** | `hsl(186, 86%, 53%)` | 色相、飽和度、亮度 |

## 複製顏色

1. 將游標定位到需要的像素上。
2. 按 <kbd>C</kbd>（不加修飾鍵）將目前顏色值複製到剪貼簿。
3. 出現短暫的確認提示，表示顏色已複製。

## 快速鍵

| 按鍵 | 操作 |
|-----|--------|
| <kbd>X</kbd> | 在 HEX、RGB 和 HSL 格式間切換 |
| <kbd>C</kbd> | 將目前顏色值複製到剪貼簿 |

取色器啟用時（懸停或已確認模式）這些快速鍵即可使用。它們支援跨多顯示器工作 —— 游標下的覆蓋層處理快速鍵。

## 視覺回饋

- 取色器顯示**目前顏色色塊**及其格式化值。
- 複製後短暫顯示**複製確認**。
- 格式指示器顯示目前啟用的格式。
