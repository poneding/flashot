# 設定

Flashot 的設定視窗允許您自訂應用程式的各個方面。從系統托盤圖示選單中即可存取。

設定分為三個標籤頁：**一般**、**外觀**和**快速鍵**。

## 一般

### 預設儲存目錄

設定使用**儲存**功能時截圖的存放資料夾。點選資料夾圖示瀏覽並選擇其他位置。預設為 `~/Pictures/Flashot`。

### 開機自動啟動

啟用後 Flashot 會在您登入電腦時自動啟動。Flashot 會在背景安靜執行，等待您觸發擷取。

### 自動更新

控制 Flashot 檢查和安裝更新的方式：

- **自動檢查更新** — 啟用後，Flashot 會按設定的間隔定期檢查新版本（預設每 24 小時，可在 1–168 小時間設定）。
- **允許測試版更新** — 啟用後，Flashot 會先檢查測試版/預先發行版本，若無則回退到穩定版。適合想提前體驗新功能的使用者。

## 外觀

### 語言

Flashot 支援三種語言：

| 語言 | 代碼 |
|----------|------|
| 英文 | `en` |
| 簡體中文 | `zh-CN` |
| 繁體中文 | `zh-TW` |

語言設定會影響所有介面文字，包括設定、標註工具提示和關於視窗。

### 主題

提供三種模式：

- **跟隨系統** — 跟隨作業系統外觀設定（淺色/深色）
- **淺色** — 始終淺色
- **深色** — 始終深色

### 強調色

從五種強調色中選擇，用於著色選擇框、工具列高亮和圖釘發光效果：

| 顏色 | 預設色值 |
|-------|---------|
| 青色 | `#22d3ee` |
| 玫瑰紅 | `#f43f5e` |
| 琥珀 | `#f59e0b` |
| 翡翠綠 | `#10b981` |
| 紫羅蘭 | `#8b5cf6` |

## 快速鍵

三個全域快速鍵可以自訂：

| 功能 | 預設（macOS） | 預設（Windows/Linux） | 說明 |
|----------|---------------|-----------------------|------|
| 區域擷取 | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | 打開覆蓋層進行區域選擇 |
| 全螢幕擷取 | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> | 快捷擷取 — 整個螢幕 |
| 視窗擷取 | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> | 快捷擷取 — 最前面的視窗 |

修改快速鍵：

1. 點選操作對應的**快速鍵輸入框**。
2. 在鍵盤上按下所需的組合鍵。
3. 輸入框會更新顯示您的新快速鍵。

如果兩個快速鍵衝突，Flashot 會顯示警告提示。快速鍵修改即時生效，無需重新啟動。

### 恢復預設

點選設定視窗底部的**重設**按鈕可恢復所有預設值。
