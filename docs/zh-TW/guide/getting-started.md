# 快速上手

安裝並啟動 Flashot 後（在系統托盤/選單列中尋找閃電圖示），即可開始擷取。

## 預設快速鍵

| 操作 | macOS | Windows / Linux |
|--------|-------|-----------------|
| 區域擷取 | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> |
| 全螢幕快截 | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> |
| 視窗快截 | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> |

## 第一次擷取

1. 按下 **<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>**（macOS）或 **<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>**（Windows/Linux）。
2. 畫面凍結，覆蓋層出現。
3. **點選並拖動**選擇區域，或**點選**某個視窗自動選取其邊界。
4. 放開滑鼠確認選擇。

![Flashot 擷取展示](/images/demo-01.png)

### 選擇後的操作

確認選區後，操作工具列會顯示以下選項：

- **複製** — 將截圖複製到剪貼簿（<kbd>Cmd</kbd> + <kbd>C</kbd>）
- **儲存** — 儲存為 PNG 檔案（<kbd>Cmd</kbd> + <kbd>S</kbd>）
- **圖釘** — 釘為浮動視窗
- **滾動** — 開始滾動截圖
- **關閉** — 取消（<kbd>Esc</kbd>）

您還可以透過拖拽選擇框的**手柄**來調整選區大小，或在選區內拖拽**移動**選區位置。

## 快捷擷取

快捷擷取會跳過覆蓋層，直接將結果複製到剪貼簿：

- **全螢幕擷取** — 擷取主顯示器的全部畫面
- **活動視窗擷取** — 擷取最前面的視窗

## 標註

確認選區後，標註工具列會出現在選區上方。使用它可以繪製箭頭、矩形、文字，對敏感資訊進行模糊處理等。各工具可自由切換，每個工具獨立記住自己的樣式設定。

## 取消

在擷取過程中隨時按 <kbd>Esc</kbd> 取消並返回桌面。
