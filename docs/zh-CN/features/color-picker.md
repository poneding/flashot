# 取色器

Flashot 内置取色器，让您直接从截图中采样颜色。

## 使用方法

1. 截取区域并确认选区。
2. 将光标悬停在截图区域上。
3. 取色器面板在覆盖层处于**悬停**或**已确认**模式时自动显示。
4. 移动光标查看实时颜色值更新。

## 颜色格式

取色器支持三种格式，按 <kbd>X</kbd> 键切换：

| 格式 | 示例 | 说明 |
|--------|---------|-------------|
| **HEX** | `#22d3ee` | 标准十六进制表示 |
| **RGB** | `rgb(34, 211, 238)` | 红、绿、蓝（0–255） |
| **HSL** | `hsl(186, 86%, 53%)` | 色相、饱和度、亮度 |

## 复制颜色

1. 将光标定位到需要的像素上。
2. 按 <kbd>C</kbd>（不加修饰键）将当前颜色值复制到剪贴板。
3. 出现短暂的确认提示，表示颜色已复制。

## 快捷键

| 按键 | 操作 |
|-----|--------|
| <kbd>X</kbd> | 在 HEX、RGB 和 HSL 格式间切换 |
| <kbd>C</kbd> | 将当前颜色值复制到剪贴板 |

取色器激活时（悬停或已确认模式）这些快捷键即可使用。它们支持跨多显示器工作 —— 光标下的覆盖层处理快捷键。

## 视觉反馈

- 取色器显示**当前颜色色块**及其格式化值。
- 复制后短暂显示**复制确认**。
- 格式指示器显示当前激活的格式。
