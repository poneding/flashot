# 图像调整

在保存或复制截图之前，您可以实时应用图像调整。这些调整在截取时实时预览，并会合入输出文件。

## 调整控制

从截图工具栏中打开调整面板（查找图像图标）。提供四个控制项：

### 亮度

调整图像的整体明暗程度。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 明显变亮 |
| `+1` 到 `+49` | 略微变亮 |
| `0` | 无变化 |
| `-1` 到 `-49` | 略微变暗 |
| `-50` 到 `-100` | 明显变暗 |

### 对比度

控制亮部和暗部之间的差异。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 高对比度 —— 鲜明、清晰 |
| `+1` 到 `+49` | 略微提高对比度 |
| `0` | 无变化 |
| `-1` 到 `-49` | 略微降低对比度 |
| `-50` 到 `-100` | 低对比度 —— 平淡、柔和 |

### 饱和度

控制色彩强度。

| 值 | 效果 |
|-------|--------|
| `+50` 到 `+100` | 鲜艳、过饱和的色彩 |
| `+1` 到 `+49` | 略微更生动 |
| `0` | 无变化 |
| `-1` 到 `-49` | 略微去饱和 |
| `-50` 到 `-100` | 高度去饱和 |
| **灰度开启** | 完全去饱和（覆盖滑动条） |

### 灰度

一个开关，移除图像中的所有颜色。启用时覆盖饱和度滑动条。

## 使用说明

- 调整在冻结层上**实时预览**。
- 调整同时影响**剪贴板复制**和**保存的文件**。
- 调整是**非破坏性的** —— 原始帧保留在内存中。
- 钉住的截图窗口也支持调整，以便在导出前精细调整。

## 键盘快捷键

调整功能没有直接的键盘快捷键。请使用面板上的控制项或单独重置每个滑动条。
