# 滚动截屏

滚动截屏让您可以捕捉超出可视区域的内容 —— 如长网页、文档或聊天记录 —— 通过自动拼接多帧画面实现。

## 工作原理

1. 在屏幕上**选择一个区域**。
2. 点击操作工具栏中的**滚动**按钮（注意：所选区域高度必须至少 100px）。
3. 出现半透明覆盖层，提示您**滚动内容**。
4. 开始滚动目标窗口 —— Flashot 会随着滚动自动捕捉帧。
5. 状态药丸显示当前的**帧数**和**总高度**。
6. 点击右下角的**绿色勾选按钮**（✓）完成截取。

捕捉到的帧会被拼接成一幅完整的长图，然后保存或复制。

## 滚动 Chrome 界面

滚动截屏过程中，会出现一个"滚动 Chrome"窗口，包含：

- **实时预览** — 捕捉到的帧垂直堆叠，带平滑过渡动画
- **状态药丸** — 底部中央显示 `X 帧 · Y px`
- **完成按钮** — 绿色勾选按钮结束截取
- **自动停止** — 达到最大高度时自动结束会话

## 技术细节

- 随着滚动，每隔几百毫秒捕捉一帧。
- 每帧与前一帧使用评分算法比较，检测新内容。
- 仅保留包含足够新内容的帧（去重）。
- 使用基于 NCC（归一化互相关）评分的接缝检测算法，在重叠内容中寻找最佳拼接位置。

## 提示

- **平稳滚动** — 稳定、中速的滚动产生最佳的拼接效果。
- **选择合适的区域** — 截取区域应覆盖内容区域，但如果可能，排除固定的页眉/页脚。
- 最小选择高度为 100px。
- 最大拼接高度取决于可用内存和内容。
