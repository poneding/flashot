# 图钉

**图钉**功能将截图转为始终置顶的浮动窗口。适用于：

- 在其他应用中工作时参考截图
- 并排比较多张截图
- 在演示中保持标注后的截图可见

## 钉截图

1. 截取一个区域。
2. 点击操作工具栏中的**图钉**按钮。
3. 截图在新浮动窗口中打开。

## 控制

悬停在钉住的截图上（或获取焦点）时显示控制项。控制项位于钉住窗口的右侧：

| 控制 | 操作 |
|---------|--------|
| ✏️ **编辑** | 进入标注模式，添加或修改标注 |
| 🖼️ **调整** | 打开图像调整面板 |
| 🔍 **缩放** | 打开缩放比例选择器（50%–300%） |
| ✕ **关闭** | 关闭钉住窗口 |
| 💾 **保存** | 将钉住的截图保存为 PNG |
| 📋 **复制** | 复制到剪贴板（带勾选确认动画） |

### 缩放

- **鼠标滚轮** — 向上/向下滚动以 5% 为步长放大/缩小
- <kbd>Cmd</kbd> + <kbd>+</kbd> — 放大
- <kbd>Cmd</kbd> + <kbd>-</kbd> — 缩小
- <kbd>Cmd</kbd> + <kbd>0</kbd> — 重置为 100%

当前缩放百分比短暂显示在左上角。

### 移动

在钉住的图像上点击并拖拽以重新定位窗口。

### 关闭

- 点击**关闭**按钮
- 按 <kbd>Esc</kbd>
- **双击**钉住图像上的任意位置

## 编辑钉住的截图

点击**编辑**按钮进入钉住截图的标注模式：

1. 完整的标注工具栏可用。
2. 在钉住的图像上绘制标注。
3. 按 <kbd>Esc</kbd> 或再次点击**编辑**保存并退出。

保存或复制时，标注会合入输出。标注也作为独立图层持久保存，以便后续重新编辑。

## 图钉上的图像调整

调整面板也适用于钉住的截图。调整亮度、对比度、饱和度或启用灰度 —— 效果在保存/复制时应用。

## 键盘快捷键

| 按键 | 操作 |
|-----|--------|
| <kbd>Esc</kbd> | 退出编辑模式或关闭图钉 |
| <kbd>E</kbd> | 进入标注编辑模式 |
| <kbd>Cmd</kbd> + <kbd>C</kbd> | 复制图钉到剪贴板 |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | 将图钉保存为 PNG |
| <kbd>Cmd</kbd> + 滚轮 | 放大/缩小 |
