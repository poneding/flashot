# 标注工具

确认选区后，Flashot 进入标注模式。标注工具栏出现在选定区域上方，提供 **13 种工具**用于标记截图。

## 工具一览

| 工具 | 图标 | 用途 |
|------|------|---------|
| 矩形 | `□` | 绘制描边或填充矩形 |
| 椭圆 | `○` | 绘制描边或填充椭圆 |
| 箭头 | `→` | 带可调箭头大小的方向指示 |
| 直线 | `—` | 可自定义描边的直线 |
| 画笔 | `✏️` | 可变粗细的自由手绘 |
| 文字 | `T` | 添加文字标签，支持字体、大小和颜色选择 |
| 序号标记 | `#` | 带编号的标注，适合按顺序标记 |
| 高亮 | `🖍️` | 半透明高亮 —— 两种模式：自由手绘倾斜或直线 |
| 马赛克 | `●` | 对敏感信息（文字、人脸、数字）进行像素化模糊处理 |
| 放大镜 | `🔍` | 放大截图的某个区域以查看细节 |
| 聚光灯 | `🔦` | 除焦点区域（圆形或矩形）外全部变暗 |
| 测量 | `📏` | 测量像素距离并显示尺寸 |
| 橡皮擦 | `🧹` | 点击即可删除单个标注对象 |

每个工具独立记忆自己的样式设置（描边颜色、填充颜色、不透明度、粗细、字体）—— 每个工具最后使用的样式会被记住。

## 绘制流程

1. 点击标注工具栏中的图标**选择工具**。
2. 在截图上**点击并拖动**创建标注对象。
3. **松开**将其放置在图像上。
4. **点击已有对象**选中它，在工具栏下方出现的属性面板中调整属性。

标注工具栏是**可拖动的** —— 抓住左侧的拖拽手柄即可将其移动到屏幕上任意位置。

## 属性面板

选中标注对象后，属性面板显示其可调属性：

| 属性 | 适用工具 |
|----------|------------|
| 描边颜色 | 除橡皮擦外的所有工具 |
| 描边宽度 | 矩形、椭圆、箭头、直线、画笔 |
| 填充颜色 | 矩形、椭圆 |
| 填充不透明度 | 矩形、椭圆 |
| 字体 | 文字 |
| 字号 | 文字 |
| 箭头大小 | 箭头 |
| 模糊半径 | 马赛克（导出时应用高斯模糊） |
| 标记编号 | 序号标记 |

## 撤销 / 重做

- **撤销** — <kbd>Cmd</kbd> + <kbd>Z</kbd> / <kbd>Ctrl</kbd> + <kbd>Z</kbd>
- **重做** — <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> / <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd>

## 删除

选中对象后按 **<kbd>Delete</kbd>** 或 **<kbd>Backspace</kbd>**，或使用橡皮擦工具点击对象将其移除。

## 工具提示

每个工具按钮悬停时显示描述性提示。工具切换没有键盘快捷键 —— 通过点击工具栏选择工具。

## 标注持久化

**复制**或**保存**时，标注会被栅格化并合并到输出图像中。对于钉住的截图，标注会作为独立图层保存，可在图钉编辑模式中重新编辑。
