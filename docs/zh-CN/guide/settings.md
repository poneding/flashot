# 设置

Flashot 的设置窗口允许您自定义应用的各个方面。从系统托盘图标菜单中即可访问。

设置分为三个标签页：**通用**、**外观**和**快捷键**。

## 通用

### 默认保存目录

设置使用**保存**功能时截图的存放文件夹。点击文件夹图标浏览并选择其他位置。默认为 `~/Pictures/Flashot`。

### 开机自启

启用后 Flashot 会在您登录电脑时自动启动。Flashot 会在后台安静运行，等待您触发截取。

### 自动更新

控制 Flashot 检查和安装更新的方式：

- **自动检查更新** — 启用后，Flashot 会按设定的间隔定期检查新版本（默认每 24 小时，可在 1–168 小时间配置）。
- **允许测试版更新** — 启用后，Flashot 会先检查测试版/预发布版本，若无则回退到稳定版。适合想提前体验新功能的用户。

## 外观

### 语言

Flashot 支持三种语言：

| 语言 | 代码 |
|----------|------|
| 英语 | `en` |
| 简体中文 | `zh-CN` |
| 繁体中文 | `zh-TW` |

语言设置会影响所有界面文字，包括设置、标注工具提示和关于窗口。

### 主题

提供三种模式：

- **跟随系统** — 跟随操作系统外观设置（浅色/深色）
- **浅色** — 始终浅色
- **深色** — 始终深色

### 强调色

从五种强调色中选择，用于着色选择框、工具栏高亮和图钉发光效果：

| 颜色 | 预览色值 |
|-------|---------|
| 青色 | `#22d3ee` |
| 玫瑰红 | `#f43f5e` |
| 琥珀 | `#f59e0b` |
| 翡翠绿 | `#10b981` |
| 紫罗兰 | `#8b5cf6` |

## 快捷键

三个全局快捷键可以自定义：

| 功能 | 默认（macOS） | 默认（Windows/Linux） | 说明 |
|----------|---------------|-----------------------|------|
| 区域截取 | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | 打开覆盖层进行区域选择 |
| 全屏截取 | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> | 快捷截取 — 整个屏幕 |
| 窗口截取 | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> | 快捷截取 — 最前面的窗口 |

修改快捷键：

1. 点击操作对应的**快捷键输入框**。
2. 在键盘上按下所需的组合键。
3. 输入框会更新显示您的新快捷键。

如果两个快捷键冲突，Flashot 会显示警告提示。快捷键修改实时生效，无需重启。

### 恢复默认

点击设置窗口底部的**重置**按钮可恢复所有默认值。
