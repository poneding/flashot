# 快速上手

安装并启动 Flashot 后（在系统托盘/菜单栏中寻找闪电图标），即可开始截取。

## 默认快捷键

| 操作 | macOS | Windows / Linux |
|--------|-------|-----------------|
| 区域截取 | <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> | <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd> |
| 全屏快截 | <kbd>Opt</kbd> + <kbd>F</kbd> | <kbd>Win</kbd> + <kbd>F</kbd> |
| 窗口快截 | <kbd>Opt</kbd> + <kbd>W</kbd> | <kbd>Win</kbd> + <kbd>W</kbd> |

## 第一次截取

1. 按下 **<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>**（macOS）或 **<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>A</kbd>**（Windows/Linux）。
2. 屏幕冻结，覆盖层出现。
3. **点击并拖动**选择区域，或**单击**某个窗口自动选中其边界。
4. 松开鼠标确认选择。

![Flashot 截取演示](/images/demo-01.png)

### 选择后的操作

确认选区后，操作工具栏会显示以下选项：

- **复制** — 将截图复制到剪贴板（<kbd>Cmd</kbd> + <kbd>C</kbd>）
- **保存** — 保存为 PNG 文件（<kbd>Cmd</kbd> + <kbd>S</kbd>）
- **图钉** — 钉为浮动窗口
- **滚动** — 开始滚动截屏
- **关闭** — 取消（<kbd>Esc</kbd>）

您还可以通过拖拽选择框的**手柄**来调整选区大小，或在选区内拖拽**移动**选区位置。

## 快捷截取

快捷截取会跳过覆盖层，直接将结果复制到剪贴板：

- **全屏截取** — 截取主显示器的全部画面
- **活动窗口截取** — 截取最前面的窗口

## 标注

确认选区后，标注工具栏会出现在选区上方。使用它可以绘制箭头、矩形、文字，对敏感信息进行模糊处理等。各工具可自由切换，每个工具独立记住自己的风格设置。

## 取消

在截取过程中随时按 <kbd>Esc</kbd> 取消并返回桌面。
