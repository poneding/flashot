# Pin

The **Pin** feature turns a screenshot into a floating window that stays on top of other applications. This is useful for:

- Referencing a screenshot while working in another app
- Comparing multiple screenshots side by side
- Keeping annotated screenshots visible during presentations

## Pinning a Screenshot

1. Capture a region.
2. Click the **Pin** button in the action toolbar.
3. The screenshot opens in a new floating window.

## Controls

Pinned screenshots show controls when you hover over them (or when focused). Controls are located on the right side of the pinned window:

| Control | Action |
|---------|--------|
| ✏️ **Edit** | Enter annotation mode to add or modify annotations |
| 🖼️ **Adjustments** | Open image adjustment panel |
| 🔍 **Zoom** | Opens a zoom level picker (50%–300%) |
| ✕ **Close** | Close the pinned window |
| 💾 **Save** | Save the pinned screenshot as PNG |
| 📋 **Copy** | Copy to clipboard (with checkmark confirmation) |

### Zoom

- **Mouse wheel** — Scroll up/down to zoom in/out in 5% increments
- <kbd>Cmd</kbd> + <kbd>+</kbd> — Zoom in
- <kbd>Cmd</kbd> + <kbd>-</kbd> — Zoom out
- <kbd>Cmd</kbd> + <kbd>0</kbd> — Reset zoom to 100%

The current zoom percentage is shown briefly in the top-left corner when changing.

### Moving

Click and drag anywhere on the pinned image to reposition the window.

### Closing

- Click the **Close** button
- Press <kbd>Esc</kbd>
- **Double-click** anywhere on the pinned image

## Editing Pinned Screenshots

Click the **Edit** button to enter annotation mode on a pinned screenshot:

1. The full annotation toolbar becomes available.
2. Draw annotations on the pinned image.
3. Press <kbd>Esc</kbd> or click **Edit** again to save and exit.

When you save or copy, annotations are baked into the output. They're also persisted as a separate layer so you can re-edit later.

## Image Adjustments on Pins

The adjustments panel works on pinned screenshots too. Adjust brightness, contrast, saturation, or enable grayscale — the effects are applied on save/copy.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| <kbd>Esc</kbd> | Exit edit mode or close pin |
| <kbd>E</kbd> | Enter annotation edit mode |
| <kbd>Cmd</kbd> + <kbd>C</kbd> | Copy pin to clipboard |
| <kbd>Cmd</kbd> + <kbd>S</kbd> | Save pin as PNG |
| <kbd>Cmd</kbd> + Scroll | Zoom in/out |
