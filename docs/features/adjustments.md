# Image Adjustments

Before saving or copying a screenshot, you can apply real-time image adjustments. These adjustments are applied visually during capture and baked into the output file.

## Adjustment Controls

Open the adjustments panel from the screenshot toolbar (look for the image icon). It provides four controls:

### Brightness

Adjust the overall lightness of the image.

| Value | Effect |
|-------|--------|
| `+50` to `+100` | Significantly brighter |
| `+1` to `+49` | Slightly brighter |
| `0` | No change |
| `-1` to `-49` | Slightly darker |
| `-50` to `-100` | Significantly darker |

### Contrast

Control the difference between light and dark areas.

| Value | Effect |
|-------|--------|
| `+50` to `+100` | High contrast — punchy, defined |
| `+1` to `+49` | Slightly more contrast |
| `0` | No change |
| `-1` to `-49` | Slightly flatter |
| `-50` to `-100` | Low contrast — washed out, muted |

### Saturation

Control color intensity.

| Value | Effect |
|-------|--------|
| `+50` to `+100` | Vibrant, oversaturated colors |
| `+1` to `+49` | Slightly more vivid |
| `0` | No change |
| `-1` to `-49` | Slightly desaturated |
| `-50` to `-100` | Heavy desaturation |
| **Grayscale on** | Fully desaturated (overrides slider) |

### Grayscale

A toggle that removes all color from the image. When enabled, it overrides the saturation slider.

## Usage Notes

- Adjustments are **previewed live** on the frozen layer.
- Adjustments affect both the **clipboard copy** and the **saved file**.
- Adjustments are **not destructive** — the original frame is preserved in memory.
- The pinned screenshot window also supports adjustments for fine-tuning before export.

## Keyboard Shortcuts

There are no direct keyboard shortcuts for adjustments. Use the panel controls or reset each slider individually.
